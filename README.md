# Exercice GitHub

Petit projet pour apprendre les bases de Git et GitHub.

## Description
Ce dépôt contient des exercices et des fichiers d'exemple pour la session 1 — Introduction à Git. Objectifs : initier un dépôt, faire des commits, créer des branches et pousser vers GitHub.

## Installation / Prérequis
- Git installé : https://git-scm.com/
- (Optionnel) Node.js si des scripts sont utilisés.
